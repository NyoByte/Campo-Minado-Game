Contexto:

Hemos realizado la creación de un juego en una página web como trabajo del curso de Programación Web de la carrera de Ing. Sistemas en la Universidad de Lima. El objetivo del desarrollo del juego es llegar a usuarios de todo el mundo con esta aplicación web que cuenta con visualización y gráficos mejorados e interactivos. Se realizó el proyecto con metodología ágil: Scrum.
